# 💻 TechSphere: Plataforma de Eventos de Tecnologia

Este projeto simula uma plataforma web para um evento de tecnologia, desenvolvido como parte do projeto final das disciplinas de Gestão de Tecnologia da informação e Análise de Sistemas.

## 🛠️ Tecnologias Utilizadas

- **Front-end:** HTML5, CSS3, JavaScript (Vanilla).
- **Back-end:** Node.js, Express.
- **Arquitetura:** API RESTful básica com arquivos estáticos.

## 🚀 Como Executar o Projeto

O projeto agora roda em um servidor Node.js e não pode mais ser aberto clicando direto no arquivo HTML. Siga os passos abaixo:

1. Certifique-se de ter o **Node.js** instalado na sua máquina.
2. Clone o repositório e abra o terminal na pasta raiz do projeto.
3. Instale as dependências do servidor:
   `npm install`
4. Inicie o servidor com o comando:
   `node server.js`
5. Abra o navegador e acesse: `http://localhost:3000`
